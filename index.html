📊 Descriptive Statistics

Mean (Average)

x̄ = Σx ÷ n

Add up all the values, then divide by how many there are.

x̄ = meanΣx = sum of all valuesn = count

Sample Standard Deviation

s = √[ Σ(x − x̄)² ÷ (n−1) ]

Measures how spread out values are from the mean. Use n−1 for samples.

s = std devx̄ = meann = sample size

IQR (Interquartile Range)

IQR = Q3 − Q1

The spread of the middle 50% of data. Not affected by outliers.

Z-Score

z = (x − μ) ÷ σ

How many standard deviations above or below the mean a value is.

x = valueμ = meanσ = std dev
🎲 Probability

Basic Probability

P(A) = favourable ÷ total outcomes

The fraction of outcomes that are the event you want.

Complement Rule

P(not A) = 1 − P(A)

Probability of something NOT happening = 1 minus it happening.

Addition Rule

P(A or B) = P(A) + P(B) − P(A and B)

Subtract the overlap so you don't count it twice.

Conditional Probability

P(A|B) = P(A and B) ÷ P(B)

Probability of A, given that B has already happened.

🔬 Inferential Statistics

Standard Error

SE = s ÷ √n

How much the sample mean is expected to vary from the true mean. Gets smaller with more data.

95% Confidence Interval

CI = x̄ ± 1.96 × SE

The range likely to contain the true population mean 95% of the time.

Pearson Correlation (r)

−1 ≤ r ≤ +1

+1 = perfect positive, −1 = perfect negative, 0 = no linear relationship.
